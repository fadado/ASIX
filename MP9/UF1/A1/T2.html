Conceptes bàsics del protocol HTTP

Què és HTTP?
Hypertext Transfer Protocol, un protocol client/servidor de la capa d’aplicació.
Què és una transacció HTTP?
Un intercanvi de missatges entre client i servidor.
Què és un missatge HTTP?
La informació enviada de client a servidor o viceversa.
Què és una petició HTTP?
El missatge enviat de client a servidor.
Què és una resposta HTTP?
El missatge enviat de servidor a client.
Com són els missatges HTTP?
Consten d’una primera línia especial, seguida de zero o més capçaleres, una línia en blanc i un cos del missatge opcional.
Com és la primera línia de les peticions?
MÈTODE URL VERSIÓ
Com és la primera línia de les respostes?
VERSIÓ ESTAT TEXT
Quins mètodes principals s’usen a la Web?
GET i POST.
Quan usa un navegador el mètode GET?
Quan així ho indica l’atribut method de l’element form i en enllaços normals (atribut href d’element a i link, atribut src d’element img i script.).
Quan usa un navegador el mètode POST?
Quan així ho indica l’atribut method de l’element form.
Com són els salts de línia del protocol HTTP?
Consten de la parella de caràcters ASCII CR i LF.
Com són les capçaleres HTTP?
Consten d’un nom seguit del caràcter dos punts, espais en blancs i una cadena de caràcters. Una línia que comença amb espais en blanc és continuació de l’anterior.
Què és el cos d’un missatge HTTP?
Una cadena de bytes opaca al protocol, podríem dir que per aquest és de tipus binari.
Com podem interpretar el contingut del cos?
Per la capçalera Content-Type.
Com sabem la quantitats de bytes del cos?
Per la capçalera Content-Length.
Si el cos és de tipus text, quins finals de línia pot usar?
El protocol HTTP no diu res al respecte.
Quan proporcionem a un navegador la URL d’una pàgina a visitar, quina és la primera operació que ha de realitzar?
Consultar el DNS per convertir de nom de host a IP.
Com sap un servidor el nom de host emprat per l’usuari del client?
Capçalera HOST si és que el client es comporta correctament i la posa en la petició.
Quines son les tres parts principals de les URLs?
Protocol, host i recurs.
Quantes peticions ha de fer una navegador per mostrar una pàgina HTML?
  • Una per la pàgina HTML.
  • Una per cada element img referenciat dins la pàgina.
  • Una per cada full d’estil CSS referenciat dins la pàgina.
  • Una per cada script JavaScript referenciat dins la pàgina.
  • Altres…
