Processament de continguts amb Pandoc

Pandoc

Pandoc permet generar pàgines de manual, documents HTML, PDF o ODT, etc.

Per automatitzar el processament usarem scripts del Shell, que cridaran repetidament a pandoc per generar pàgines de manual, pàgines XHTML, etc.

Enllaços recomanats

  • Us de pandoc
  • Plantilles de pandoc
  • Blocs de metainformació a Pandoc
  • Descripció de YAML

Pràctiques

Aquesta pràctica continua l’anterior: projecte a GitLab amb dos o tres documents de MarkDown agafats directament del repositori de Docker.

  • Estudia el sistema de plantilles de Pandoc, i crea una per XHTML a partir de la generada amb pandoc -D html.
  • Estudia la sintaxi de YAML
  • Fes un script del Shell que generi pàgines XHTML pels documents de Docker que has posat en aquest projecte. Comenta apropiadament aquest script.
  • Fes que el script validi les pàgines XHTML generades.
  • Els fitxers generats no s’han d’afegir al repositori de Git (posa els seus noms en el fitxer .gitignore).
  • No deixis de pujar el repositori a GitLab.
  • Et farà falta adaptar els documents que has agafat de Docker. Observa aquests detalls:
    • Els documents tenen metainformació al principi de tot, però no amb la sintaxi pròpia de Pandoc. Et caldrà per tant fer canvis: mira de conservar la metainformació dins d’un comentari XML.
    • Et caldrà veure els noms que la plantilla espera trobar dins la metainformació del document.
    • Modifica la plantilla per que tingui un menú amb enllaços a totes les pàgines generades.
    • Modifica la plantilla per que tingui CSS personalitzat.
    • Quina mena d’HXTML es genera? Podem fer que sigui estricte?
  • Extra: modifica el script per generar també pàgines de manual en el format de man(1).
