Publicació de continguts amb Jekyll

Integració de Jekyll i GitHub Pages

Una vegada hem generat un lloc amb Jekyll podem pensar que el que cal fer per publicar-lo és posar a Internet el contingut del directori _site… Doncs no! Tenim una solució millor: usar git push i deixar que sigui GitHub qui generi el contingut del directori _site del a partir dels documents font. Aquest és el comportament per defecte de GitHub Pages i no cal configurar res per que aquesta màgia succeeixi.

Com que localment hauràs executat Jekyll, et caldrà posar aquest fitxer .gitignore en el teu repositori:

	_site
	.sass-cache
	.jekyll-metadata
